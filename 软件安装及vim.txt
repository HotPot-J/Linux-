一.yum （一般下载需要权限： su - root 该命令则获得了管理员权限）
   1.包管理器
   
   2.安装： yum install -y(该选项为yes的意思,可省略但随后需要用Y确认是否安装)  [包名 即要下载的软件名] 例子：下载一个tree  yum install tree
   
   3.卸载： yum remove [包名]  例子： 卸载tree   yum remove tree
   
   4.查看： yum list 
     eg: | grep tree 
	 
二. 安装了lrzsz软件的用处
   1.rz： 该命令是从windows操作系统下拷贝一个文件到linux操作系统下
   2.sz+文件名 ： 该命令是从linux操作系统下拷贝一个文件到windows操作系统下
   
   (xshell的xftp功能也是支持linux和win之间的文件传输，比rzsz做得好的地方是支持文件夹传输   （该功能需要单独下载）)
   
三. vim(编辑器)
    1.vim [filename]
	2.vim的三种模式
	  a.普通模式 
	          光标的移动： 上下左右键 或 h(光标向左移动) j(光标向下移动) k(光标向上移动) i(光标向右移动)
        	  i命令到insert模式在当前光标的前面进行插入  esc回到普通模式
			  I（shift+i）命令到行首进行插入
			  o命令 另起一行并进入插入模式
			  O命令 上走一行并进入插入模式
			  a命令 进入insert模式 在当前光标的后面进行插入
			  A命令 光标移动到改行末尾并进入插入模式
			  w命令(光标跳到下一个字的自首)
			  b命令(光标跳到上一个字字首)
			  e命令(光标跳到下一个字字尾)
			  进入底行模式： :  esc回到普通模式
			  
	  b.插入模式
	  
	  c.底行模式
	          ：w命令 保存   ：！q 强制退出 ：wq保存退出 ：wq！强制保存退出
			  ：set nu 显示行号
			  ：set nonu 不现实行号
			  ：%s/[deststr]/[srcstr]/g -->将全文的deststr全部替换成srcstr
			  ：s/[deststr]/[srcstr]/g -->将光标所在行的所有deststr全部替换成srcstr
			  ：[num1,num2]s/[deststr]/[srcstr]/g -->从num1到num2行的deststr全部替换成srcstr
			  
	3.移动
	       gg： 回到当前文本的头部
		   G：  移动到当前文本的尾部
	  删除
	       x： 删除一个字符
		   X： 删除当前光标的前一个字符
	  [num]X:  删除当前光标的前num个字符
		   dd：删除当前光标所在的那一行   使用p命令：在当前光标下一行中粘贴使用dd命令剪切的内容
	  [num]dd: 删除当前光标包括其的共num行
	  复制
	       yy：复制当前光标的这一行
	  [num]yy: 复制多行数据
	       yw: 拷贝包光标所在字
	  [num]yw: 拷贝从包含光标所在字的num个字
	  小写 p ：向光标的下一行取粘贴
	  大写 P ：向光标的上一行取粘贴
	  
	  替换   ：
	           r： 在普通模式  替换当前光标所在字符
			       流程，在普通模式下，按一个r表示替换光标所在的字符，接下来，按想要替换成的字符
			   R： 替换多个字符，会先切换到replace模式，一直替换，如果替换完成了，按esc返回到普通模式  
	  撤销   ：
	           u： 撤回上一次的动作（若上一次动作做错了 按一下u就又回来了）
		  ctrl+r： 反撤销  u回来了 然后后悔了 把u回来的删掉
	  更改   ：
	          cw： 更改当前光标所在的单词
		 c[num]w:  更改多个字符
	       
   跳到某一行：
          [num]G:  跳到光标某一行
